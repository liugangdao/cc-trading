# Claude Code 分析 Prompt 使用指南

## 快速开始

1. 在 Claude Code 中打开项目目录
2. 复制对应的 Prompt 内容
3. 粘贴到对话框并发送
4. Claude Code 会自动读取 `trading-data/` 目录中的 JSONL 文件进行分析

## 可用 Prompts

### 风险评估（risk-assessment.md）

用于评估当前持仓的风险状况。适用场景：
- 想了解当前总体风险敞口
- 检查仓位集中度
- 评估单个仓位的风险回报比
- 获取风险预警和优化建议

### 交易优化（trade-optimization.md）

用于分析历史交易表现并获得优化建议。适用场景：
- 想了解整体交易表现（胜率、盈亏比）
- 分析不同品种和市场类型的表现差异
- 了解止损止盈策略的有效性
- 优化交易习惯和模式

## 自定义分析

你也可以直接向 Claude Code 提问，无需使用预设的 Prompt。例如：

- "分析我在 BTC/USDT 上的交易表现"
- "计算我本月的总盈亏"
- "找出我最常犯的交易错误"
- "比较我在加密货币和外汇市场的表现"
- "统计我的平均持仓时长"
- "分析哪些交易品种的胜率最高"

Claude Code 会自动读取相关数据并提供分析。

## 注意事项

- 确保 `trading-data/` 目录中有交易数据
- JSONL 文件格式为 `trades-YYYY-MM.jsonl`
- Claude Code 会自动处理多个月份的数据
- 分析结果基于现有数据，数据越多分析越准确
