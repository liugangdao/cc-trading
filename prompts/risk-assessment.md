# 交易风险评估

请分析 `trading-data/` 目录中的交易数据，重点关注当前未平仓位的风险状况。

## 分析要求

1. **总体风险敞口**
   - 计算所有未平仓位的总保证金
   - 计算最大可能损失（所有止损触发）
   - 评估风险敞口占总资金的比例

2. **单个仓位风险**
   - 列出每个未平仓位的风险回报比 (Risk/Reward Ratio)
   - 计算: RR = (TakeProfit - OpenPrice) / (OpenPrice - StopLoss)
   - 标注风险回报比 < 2 的仓位

3. **仓位集中度**
   - 按市场类型统计仓位分布
   - 按交易品种统计仓位分布
   - 标注单一品种或市场占比 > 40% 的情况

4. **风险预警**
   - 是否有仓位未设置止损或止盈（理论上不应该有）
   - 是否有仓位的止损距离过大（> 5%）
   - 是否有过度集中的风险

## 输出格式

请以清晰的结构化格式输出，包括：
- 风险总览（数字和百分比）
- 仓位明细表
- 风险预警列表
- 优化建议
